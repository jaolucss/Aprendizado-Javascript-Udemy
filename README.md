# 🚀 Aprendizado JavaScript - Udemy

Este repositório foi criado para armazenar todos os códigos, exercícios e projetos desenvolvidos por mim durante o curso de JavaScript na plataforma da Udemy. O objetivo principal é praticar a sintaxe da linguagem, lógica de programação e acompanhar minha evolução no aprendizado.

## 📂 Estrutura do Repositório

O repositório está organizado por duas pastas principais onde contem os arquivos JS, uma pasta seria com contéudo das aulas, exemplos de como usar variáveis e mostrando a sintaxe da linguagem, a outra seria as respostas dos exercícios.
